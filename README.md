# bt3-progress-chara-param-editor
A modding tool made for the [DBZ League](https://www.youtube.com/@DragonBallZLeague) that edits the contents of the 00_progress_chara_param.dat file, either directly or through the file it comes from (which is resident_chara_param.pak).
